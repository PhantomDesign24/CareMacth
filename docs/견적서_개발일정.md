| 구분 | 상세 내용 | 예상 공수 |

## 공통 포함 기능 상세

### 핵심 기능
1. 회원가입 (일반/카카오/네이버 간편가입)
2. 간병 요청 생성 (의료행위 금지 동의 필수)
3. 자동 매칭 알고리즘 (거리+경력+리뷰+재고용률-취소율)
4. 간병인 지원 및 보호자 선택
5. 자동 계약 생성 (의료행위 금지 조항 포함)
6. 에스크로 결제 (토스페이먼츠)
7. 간병 진행 (출퇴근 체크, 간병일지, 사진)
8. 평가/리뷰 시스템
9. 연장 기능 (3일전/1일전 알림)
10. 간병 교육 시스템 (수강 80% 수료)
11. 관리자 시스템 (승인, 정산, 통계, 블랙리스트)
12. 채널톡 + 카카오톡 채널 연동
13. 간병보험 서류 신청

### 결제/정산 구조
- 선결제 에스크로 (무통장 VAT별도, 카드 VAT별도, 직접결제)
- 종료 후 익일 정산: 간병비 - (플랫폼 수수료 + 3.3% 세액)
- 1:1 / 가족 간병 수수료 별도 (% + 원 단위 관리자 수정 가능)
- 장기 간병 주 단위 자동 결제
- 일할 계산 자동화
- 추가 비용 요청 → 보호자 승인 → 결제
